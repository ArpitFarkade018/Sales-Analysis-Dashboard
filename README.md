Supermarket Retail Sales Analytics Dashboard (Excel)

Project Overview

This project analyzes supermarket retail sales data using Microsoft Excel. The dashboard provides insights into sales performance, profitability, customer segments, product categories, and regional trends.

Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- KPI Cards

Key Metrics

- Total Sales
- Total Profit
- Total Orders
- Profit Margin
- Profit per Order

Dashboard Features

- Interactive filtering using slicers
- Regional sales analysis
- Monthly sales trends
- Category-wise performance
- Segment-wise sales distribution
- Sub-category profitability analysis

Key Insights

- West region generated the highest sales revenue.
- Technology category was the most profitable.
- Consumer segment contributed the largest share of revenue.
- Identified loss-making product segments for business improvement.
